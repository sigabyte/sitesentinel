Report boundary

This report is read-only. It summarizes persisted lifecycle output and links back to existing traceability pages.

It does not collect evidence, normalize evidence, generate findings, evaluate risks, generate remediation recommendations, generate trust assessments, or modify comparison output.

Persisted advisory remediation recommendations are included as read-only report output. The same report view model is intended to support a future PDF renderer without triggering recommendation generation.
